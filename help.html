2D Shape Editor Help

Welcome to the 2D Shape Editor Help page! Below you will find detailed instructions on how to use each feature.

Shapes Toolbar

Select a shape from the toolbar to start drawing. Click on a shape icon, then click on the canvas to place the shape. You can move, resize, and rotate the shape using the handles that appear under ShapeInfo tab when the shape is selected.

Export Options

You can save your canvas as different file formats. Use the 'Export' toolbar to select the desired format and follow the prompts to save your file into (JSON, JPEG, PDF, LATEX).

Undo/Redo Actions

Use the 'Undo' and 'Redo' buttons to reverse or reapply your last changes. Keyboard shortcuts are also available: Ctrl + Z for undo and Ctrl + Y for redo.

Copy/Paste Shapes

To copy a shape, select it and press the 'Copy' button or Ctrl + C. To paste, press the 'Paste' button or Ctrl + V. The pasted shape will appear slightly offset from the original.

Grid Functionality

Toggle the grid view for easier alignment of shapes. To enable snap-to-grid, click the 'Toggle Snap to Grid' button, which will align shapes to the nearest grid line when moved. You can pres Ctrl+G to toggle grid.

New Diagram

To start a new diagram, click the 'New Canvas' button. You will be prompted to save your current work if there are unsaved changes. You can press Ctrl+Space for new canvas.

Error Handling

If an error occurs, a modal will appear with the error message. Make sure to read the message carefully to understand what went wrong and how to fix it.

Advanced Shape Manipulation

Rotate and scale shapes with precision. Use the rotation handles for manual adjustments or specify the rotation angle in the properties panel for exact alignment.

Keyboard Shortcuts

Maximize your efficiency with these shortcuts:

  • Ctrl + SpaceBar: New Canvas
  • Delete/Backspace: Remove selected shapes
  • Ctrl + S: Save your work as a new JSON file
  • Ctrl + Z: Undo last action
  • Ctrl + Y: Redo last action
  • Ctrl + C: Copy selected shape
  • Ctrl + V: Paste copied shape
  • Ctrl + G: Toggle grid view

Troubleshooting Common Issues

Experiencing slowdowns? Clear your canvas or reduce the number of active layers. Ensure you are using a supported browser for optimal performance.

Saving and Loading Work

You can save your work in different formts and can load any other work which is in JSON format.

Task Bar

Use TaskBar feature to track and check all the features implemented as part of this project.
